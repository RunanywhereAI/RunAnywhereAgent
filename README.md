# RunanywhereAI — AI Coding Agent

AI-powered coding agent backed by Claude on AWS Bedrock.

## Install (Mac / Linux)

```bash
curl -fsSL https://raw.githubusercontent.com/OWNER/agent/main/install.sh | bash -s -- YOUR_TOKEN
```

## Install (Windows PowerShell)

```powershell
irm https://raw.githubusercontent.com/OWNER/agent/main/install.ps1 | iex
```

## Usage

```bash
opencode
```

That's it. Opens an interactive coding agent in your terminal.

## What You Get

- Claude Sonnet 4 — primary coding model
- Claude Haiku 4 — fast model for quick tasks
- File editing, bash execution, code search, and more
- Works in any project directory
